**Reality:** Go 1.25.5 IS the current stable version (verified via go.dev API).

---

### BUG 3: Justfile _rebuild-bib Broken Dependency ✅ FIXED
**Location:** `Justfile:195`

**Problem:** `_rebuild-bib` referenced nonexistent `build` recipe.

**Fix:** Changed to `build-aurora`.

---

### BUG 4: scrcpy Installation Lost on ostree Systems ✅ FIXED
**Location:** `files/scripts/install-scrcpy.sh`

**Problem:** Script installed to `/opt/scrcpy/` but on ostree/bootc systems, `/opt` is a symlink to `/var/opt` that gets restored after build. Files written to `/opt` during build are lost.

**Root cause:** `00-filesystem.sh` converts `/opt` symlink to directory during build, but base image restores the symlink.

**Fix:** Changed installation path from `/opt/scrcpy` to `/usr/share/scrcpy` (immutable, persisted in ostree).

---

### BUG 5: uv Tools Dangling Symlinks ✅ FIXED
**Location:** `files/scripts/install-uv.sh`, `files/system/etc/profile.d/tanzanite-dev.sh`

**Problem:** uv tools installed to `/var/cache/uv-tools` which doesn't persist on ostree. Symlinks in `/usr/local/bin/` (ruff, black, mypy, pytest, ipython) pointed to nonexistent targets.

**Fix:** Changed `UV_TOOL_DIR` from `/var/cache/uv-tools` to `/usr/share/uv-tools`.

---

### BUG 6: Gradle Wrapper Pre-caching Wasted ✅ FIXED  
**Location:** `files/scripts/install-gradle.sh`

**Problem:** Downloaded 4 gradle wrapper ZIPs to `/var/cache/gradle` during build, but /var is not persisted. Wasted bandwidth and build time.

**Fix:** Removed wrapper pre-caching. Users download on first use (normal Gradle behavior).

---

### BUG 7: 00-filesystem.sh /opt Fix Misunderstood ✅ CORRECTED
**Location:** `files/scripts/00-filesystem.sh`

**Initial mistake:** I removed the /opt fix without reading TEAM_005/009 files which explained it's needed for BUILD compatibility.

**Actual situation:**
- /opt fix IS needed during BUILD (per TEAM_005 line 162, TEAM_009 line 32)
- But files in /opt won't persist to RUNTIME on ostree (different issue)

**Correction:** Restored /opt fix with added warning comment about runtime behavior.

## OSTREE PATH RULES (for future reference)

| Path | Persists from build? | Use for |
|------|---------------------|---------|
| `/usr/*` | ✅ YES | Binaries, tools, installed software |
| `/usr/local/*` | ✅ YES | Custom binaries |
| `/usr/share/*` | ✅ YES | Application data |
| `/var/*` | ❌ NO | Runtime caches only |
| `/opt` | ❌ NO (symlink to /var/opt) | DO NOT USE |
